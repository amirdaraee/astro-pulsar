Code Blocks

Here’s a Python code example:

def fibonacci(n):
    """Generate Fibonacci sequence up to n terms."""
    fib = [0, 1]
    for i in range(2, n):
        fib.append(fib[i-1] + fib[i-2])
    return fib

# Generate first 10 Fibonacci numbers
result = fibonacci(10)
print(result)
