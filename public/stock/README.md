# Stock Images

Place your blog post cover images here.

Images in this folder can be referenced in your blog post frontmatter:

```yaml
---
title: "Your Post Title"
photo: "/stock/your-image.jpg"
---
```

## Recommended Image Sizes

- **Cover Images**: 1200x630px (optimal for social media sharing)
- **Format**: JPG, PNG, or WebP
- **File Size**: Keep under 500KB for optimal performance

## Image Optimization

Consider using tools like:
- [TinyPNG](https://tinypng.com/)
- [Squoosh](https://squoosh.app/)
- [ImageOptim](https://imageoptim.com/)

to compress your images before adding them to your blog.
